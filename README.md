# Mobile Sales Analytics Dashboard

## Overview
This project analyzes mobile sales data using Python and Tableau. The dashboard provides insights into sales performance, top brands, profit distribution, regional sales, and RAM configuration trends.


## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite3
- Tableau Public
- Jupyter Notebook


## Features
- KPI Cards for Total Sales, Profit, Quantity Sold, and Brand Count
- Top 10 Brands Analysis
- Monthly Sales Trend
- Profit by Brand Treemap
- Regional Sales Analysis
- Sales Distribution by RAM Configuration
- Interactive Dashboard Filtering


## Project Workflow
1. Data Cleaning using Pandas
2. Exploratory Data Analysis (EDA)
3. Data Visualization using Matplotlib and Seaborn
4. SQL integration using SQLite3
5. Interactive Dashboard creation in Tableau


## Dashboard Features
- KPI Cards
- Top 10 Brands Analysis
- Monthly Sales Trend
- Sales by RAM Configuration
- Profit by Brand
- Region-wise Sales Analysis
- Interactive Filtering


## Skills Demonstrated
- Data Cleaning
- Data Analysis
- SQL Queries
- Data Visualization
- Dashboard Design
- Business Insights


## Key Insights
- Google and Nokia are among the top-performing brands.
- West region generated the highest sales.
- 4GB and 16GB RAM configurations contribute major sales.
- Monthly sales fluctuate across months.
